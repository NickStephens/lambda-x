CONCRETE SYNTAX FOR PCONS

Script -> (Def | Exp)+

Def -> Name {arg} = Exp

------------------------------------------

Exp -> App | Lam | Var | Let | Case | Name

App -> Exp Exp

Lam -> L Name . Exp

Var -> Name

Let -> 'Let' Def in Exp

Case -> 'Case' Exp 'of' {Exp '->' Exp}

==========================================

Name -> Uppercase followed by any number of alphanumerics
